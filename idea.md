# О проекте
Сервис по клинингу, мультистраничный сайт который четко отвечает кластеру запросов определенной тематики

## Миграция проекта
- Проект лежит в папке `ref`, проект выполнен компонентным подходом на Laravel, необходимо проект перенести на Astro, с реализацией посадочных страниц в markdown через `content` коллекции
